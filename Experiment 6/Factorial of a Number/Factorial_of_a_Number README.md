## Problem Statement 
Write a Python program to calculate the factorial of a number 
 n using loops.

## Algorithm
1. Start
2. Read integer n
3. Set fact = 1
4. if n < 0, print "Factorial not defined" and go to step 8
5. for i = 1 to n fact = fact * i
6. Print fact
7. stop
