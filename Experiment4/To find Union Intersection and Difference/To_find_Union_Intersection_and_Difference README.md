## Problem Statement
Write a Python program to perform union, intersection and difference operations on SetA and SetB.

---

## Algorithm
1. Start.  
2. Read element of A
3. Read element of B 
4. Convert input into sets
5. find union of set A and B
6. find intersection of set A and B
7. find difference of set A and B
8. Print Union
9. Print intersection
10. Print difference
11. Exit
